Features

Metric

App Metrics provides various metric types to measure things such as the rate of requests, user activity over time, measure the time taken to execute a database query, measure the amount of free memory and so on. Metrics types supported are Apdex, Gauges, Counters, Meters, Histograms and Timers.

Reporting details

There are several time series database reporting extensions which App Metrics provides as well as more generic reporters, see the Reporters menu on the left..
